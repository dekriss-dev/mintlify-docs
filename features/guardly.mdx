## Czym jest Guardly?

Guardly to rewolucyjny bot Discord stworzony przez zespół Zgłoś Scammera. Pozwala użytkownikom budować swoją reputację poprzez zbieranie vouchy i opinii od innych użytkowników.

## Główne funkcje

### System vouchy
- **Zbieranie opinii** od innych użytkowników
- **Budowanie reputacji** na serwerach Discord
- **Weryfikacja transakcji** między użytkownikami
- **Historia transakcji** z opisami

### Profil użytkownika
- **Statystyki vouchy** - liczba pozytywnych opinii
- **Historia transakcji** - ostatnie vouchy
- **Status reputacji** - zaufany/nieznany
- **Integracja z bazą** Zgłoś Scammera

### Komendy bot
- **+rep** - dodaj pozytywną opinię
- **+vouch** - alternatywna komenda vouchy
- **+v** - skrócona wersja vouchy
- **!profile** - sprawdź profil użytkownika

## Jak dodać Guardly na serwer

### Krok 1: Zaproś bota
1. Przejdź na [stronę Guardly](https://zglosscammera.pl/guardly)
2. Kliknij **"Dodaj Guardly teraz"**
3. Wybierz serwer Discord
4. Nadaj odpowiednie uprawnienia

### Krok 2: Konfiguracja
1. **Kanał vouchy** - gdzie będą pojawiać się opinie
2. **Uprawnienia** - kto może używać komend
3. **Moderacja** - kontrola nad vouchami
4. **Integracja** - połączenie z bazą danych

### Krok 3: Testowanie
1. Wypróbuj komendę `+rep @użytkownik`
2. Sprawdź czy vouch się pojawił
3. Przetestuj inne funkcje
4. Skonfiguruj dodatkowe ustawienia

## Jak używać vouchy

### Dodawanie vouchy
```
+rep @użytkownik Konto Fortnite x1, 152 PLN (BLIK)
+vouch @użytkownik Skóra CS2, szybka transakcja
+v @użytkownik Sprzedaż konta, wszystko OK
```

### Format vouchy
- **@użytkownik** - osoba, której dajesz vouch
- **Opis przedmiotu** - co zostało sprzedane/kupione
- **Kwota** - ile kosztowała transakcja
- **Metoda płatności** - jak zapłaciłeś

### Przykłady dobrych vouchy
```
+rep @sprzedawca Konto Steam z CS2, 50 PLN (PayPal)
+vouch @kupujący Skóra Fortnite, szybka dostawa
+rep @handlarz Konto Epic Games, wszystko działa
```
